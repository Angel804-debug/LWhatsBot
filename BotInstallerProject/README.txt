===========================================
 BOT WHATSAPP – INSTALAÇÃO AUTOMÁTICA
===========================================

Olá!

Obrigado por adquirir o sistema automatizado de atendimento via WhatsApp.

Este instalador configurará automaticamente todos os componentes necessários para que seu bot funcione corretamente no seu computador Windows.

-------------------------------------------
🛠️ REQUISITOS BÁSICOS:
-------------------------------------------
- Windows 10 ou superior (64 bits)
- Permissão de administrador
- Internet ativa durante a instalação
- Google Chrome já instalado (versão 138 ou compatível)

-------------------------------------------
📦 CONTEÚDO DO PACOTE:
-------------------------------------------
- BotInstaller.exe  → Instalador automático
- XAMPP Installer    → Servidor PHP + MySQL
- Banco de dados     → banco.sql
- Painel Web         → Interface com controle e histórico
- Bot (em Python)    → dist/bot.exe
- ChromeDriver       → Integrador com o navegador

-------------------------------------------
🚀 COMO INSTALAR (passo a passo):
-------------------------------------------

1. Extraia todos os arquivos do arquivo .zip em uma pasta qualquer (ex: Área de Trabalho).

2. Clique com o botão direito no arquivo **BotInstaller.exe** e selecione **"Executar como administrador"**.

3. Aguarde todo o processo de instalação (XAMPP será instalado, arquivos copiados, banco importado).

4. Ao final da instalação:
   - O bot estará pronto para rodar.
   - O painel web estará acessível em: **http://localhost/bot_php**

5. Para iniciar o bot, clique no atalho criado ou execute o arquivo


-------------------------------------------
📌 DÚVIDAS FREQUENTES:
-------------------------------------------

- 💬 **Como acesso o painel?**
Acesse: http://localhost/bot_php no navegador.
